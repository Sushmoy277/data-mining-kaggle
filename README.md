# Data Mining Project (Kaggle)

## 📌 Overview
This project was completed as part of my university assignment. It uses data mining techniques on a FIFA dataset.

## 🔍 Tasks Performed
- Data Preprocessing
- Decision Tree (Classification)
- K-Means (Clustering)
- Apriori (Association Rule Mining)

## 🛠️ Tools Used
- Python
- Pandas
- Scikit-learn
- Kaggle Notebook

## 📊 Results
- Successfully classified data using Decision Tree
- Identified clusters using K-Means
- Generated association rules using Apriori
